My Terminal-Only Development Setup (3)

2024-02-11

In this post I’m just listing some (maybe, based on my use habits) useful commands that I find myself Googling all the time. A lot of them are based on third-party plugins and some are defined in my custom config.lua so it’s definitely not recommended to take the following as your random online reference of VIM commands.

  1. Very different concepts: buffers vs windows. For example, when a vertical split view is turned on, you can shuffle through all opened buffers in both left and right views.
